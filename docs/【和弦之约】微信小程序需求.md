# 【和弦之约】微信小程序需求

## 小程序介绍

和弦之约是一款面向情侣用户的微信轻量化纪念画册小程序，以钢琴五线谱为独特视觉载体，结合 3D 仿真翻页交互，帮助用户记录、珍藏恋爱全过程关键回忆。支持自定义事件编辑、照片收录、音乐绑定，自动生成艺术化乐谱画册，同时提供图片分享、PDF 导出等实用能力，用浪漫乐谱形式留存专属情侣情感记忆，操作轻量化、画面氛围感拉满，适合长期记录与情感收藏。

## 一、小程序核心用途 / 产品定位

- 定位：微信端恋爱回忆数字乐谱小程序，以钢琴五线谱为视觉载体，承载情侣情感回忆，主打艺术化、仪式感纪念
- 核心主线：以恋爱时光轴为串联，整合事件、照片、音乐，生成可翻阅、可导出、可分享的专属数字乐谱画册
- 核心能力：自研乐谱渲染引擎，将回忆事件转化为可视化五线谱页面，音符对应回忆卡片
- 核心体验：3D 仿真翻页动效，还原实体乐谱翻阅质感，打造沉浸式恋爱纪念体验
- 延伸能力：支持单页 / 整本分享图生成、PDF 导出，满足情侣保存、传播纪念内容的核心需求

## 二、主要功能详情

### 2.1 全局基础布局

- 顶部导航栏：固定展示 Logo「和弦之约」，右侧常驻设置图标，点击展开抽屉（主题切换、关于我们、帮助中心）
- 主体内容区：全屏乐谱画册展示区，无多余边距，小程序沉浸式独占显示
- 底部操作栏：固定悬浮，包含「上一页 / 下一页」翻页按钮、中央页码指示器、右侧功能菜单（分享、导出、编辑）
- 适配规则：仅适配微信小程序移动端，响应式兼容 iPhone / 安卓主流机型，自动适配手机安全区
- 视觉基底：全局深色暖调背景，统一音乐厅氛围感，页面切换无闪屏、无白屏

### 2.2 首页功能（时光轴画册页）

**页面内容**

- 封面页：展示画册自定义标题、情侣双方名称、恋爱起始日期，搭配钢琴键装饰元素
- 内页（乐谱页）：每页独立对应一个恋爱事件，包含动态五线谱背景、曲名式事件标题、节拍式日期标注、照片音符卡片、侧边音乐信息栏
- 差异化视觉：生日专属蜡烛音符、纪念日心形音符、相恋天数数字高亮音符、日常事件标准音符样式
- 封底页：展示相恋总天数、回忆总数量、暖心结语文案，极简设计收尾

**交互说明**

- 翻页交互：移动端支持左右滑动拖拽翻页，自带物理回弹效果；也可点击底部按钮快捷翻页
- 图片交互：点击照片卡片唤起大图预览，支持缩放、左右切换同组照片
- 音乐交互：点击音乐模块触发播放，底部轻量化迷你播放器 + 进度条展示
- 页码交互：角落展示当前 / 总页码，点击弹窗快速跳转指定页面

**登录状态**

- 初始版本为本地独立应用，无需微信登录授权，所有数据本地缓存
- 预留用户登录、云端数据同步扩展接口，支持后续迭代升级

**跳转逻辑**

- 首页为小程序默认启动首页
- 功能按钮均以弹窗、抽屉层实现，无新页面硬跳转

### 2.3 我的页面功能（编辑管理页）

**未编辑状态**

- 编辑管理入口低调隐藏，不干扰画册浏览体验
- 默认仅展示乐谱阅读界面，管理面板收起隐藏

**已进入编辑状态**

- 时光轴管理：时间线罗列全部回忆事件，支持拖拽排序、新增 / 编辑 / 删除事件，修改实时生效
- 照片管理：单事件支持批量上传、自定义描述文字、手动排序
- 音乐管理：支持填写音乐信息、本地音频上传、外部链接绑定三种方式
- 特殊日期：系统自动计算纪念日、生日、相恋时长，支持手动自定义修改

### 2.4 核心操作功能（乐谱渲染 + 画册生成）

**功能操作步骤**

1. 进入编辑面板，新增或修改恋爱事件，上传照片、绑定背景音乐
2. 前端自动调用乐谱渲染引擎，实时生成标准化五线谱画册页面
3. 切换至浏览模式，通过 3D 翻页预览完整画册内容
4. 按需使用图片分享、PDF 导出功能完成内容保存与传播

**按钮交互逻辑**

- 分享按钮：一键生成高清分享海报，支持保存相册、微信好友转发
- 导出按钮：后台分片渲染 PDF 文件，实时展示导出进度
- 编辑按钮：快速切入编辑模式，精准定位当前页面事件
- 翻页按钮：限制高频点击，防止动画重叠、页面卡顿

**页面跳转规则**

- 浏览模式与编辑模式无缝抽屉切换，保留当前阅读位置
- 所有导出、分享操作浮层化处理，不离开当前页面

## 三、界面与使用体验要求

### 3.1 界面样式规范

- 设计风格融合古典乐谱美学与现代极简 UI，贴合小程序轻量化特性
- 色彩体系统一：深墨蓝背景、暖金色谱线、奶白色卡片，特殊事件专属色标区分
- 字体规范：标题衬线艺术字体，正文系统无衬线字体，兼顾美观与可读性
- 组件统一：按钮、弹窗、输入框统一圆角风格，符合微信小程序设计规范

### 3.2 交互使用体验

- 3D 翻页动效流畅跟手，滑动无延迟、回弹自然
- 图片采用缩略图预加载 + 高清渐进加载模式，弱网友好
- 所有操作结果配备轻量 Toast 提示，删除类操作增加二次确认
- 空数据状态展示专属插画 + 引导按钮，降低用户学习成本

### 3.3 性能速度要求

- 小程序首屏加载时长 ≤ 2s
- 单页乐谱渲染耗时 ≤ 500ms，多内容页面不超过 1s
- 低端机型自动降级为 2D 翻页，保障基础流畅度
- 图片、音频资源压缩处理，采用懒加载策略，控制内存占用

### 3.4 安全相关要求

- 严格限制上传文件格式与大小：图片≤10MB、音频≤20MB
- 本地用户数据加密存储，防止数据丢失与篡改
- 相册、文件读写调用微信官方授权接口，合规合法
- 前端增加基础内容过滤，规避违规内容展示

## 四、核心功能优化建议

- **优化目标**：强化乐谱视觉质感，降低用户创作门槛，提升小程序流畅度与分享传播能力

- 具体优化方案

  ：

  - 新增谱线绘制、音符入场动态特效，提升画册仪式感
  - 内置恋爱事件通用模板，一键快速创建回忆内容
  - 上线多套乐谱主题皮肤，支持用户自由切换风格
  - 新增地理位置标记功能，记录回忆发生地点
  - 丰富分享海报模板，支持自定义文案、滤镜美化
  - 增加自动保存机制，避免编辑内容意外丢失

  

## 五、设计风格

- 整体风格：古典浪漫 + 数字精致，以钢琴、乐谱、音符为核心视觉元素
- 主色调：深墨蓝主背景，暖金色（#C9A96E）为主强调色
- 辅助色：奶白色、玫瑰金、珊瑚粉、薰衣草紫分层区分事件类型
- 材质质感：仿古纸张纹理、低阴影卡片、手写风谱线，复古温柔
- 动效调性：缓入缓出轻动效，克制不浮夸，突出情感氛围
