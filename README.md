# 和弦之约 - 微信小程序

## 项目简介

和弦之约是一款面向情侣用户的微信轻量化纪念画册小程序，以钢琴五线谱为独特视觉载体，结合3D仿真翻页交互，帮助用户记录、珍藏恋爱全过程关键回忆。

## 核心特性

- **乐谱渲染引擎**：自研引擎将回忆事件转化为可视化五线谱页面
- **3D翻页效果**：仿真翻页动效，还原实体乐谱翻阅质感
- **事件管理**：支持自定义事件编辑、照片收录、音乐绑定
- **艺术化展示**：自动生成艺术化乐谱画册，不同事件类型有差异化视觉
- **分享导出**：支持图片分享、PDF导出功能

## 主要功能模块

### 1. 首页（时光轴画册）
- 封面页展示画册标题、情侣信息
- 内页展示恋爱事件，包含动态五线谱、照片音符卡片
- 支持左右滑动翻页，物理回弹效果
- 图片预览、音乐播放功能

### 2. 我的页面（编辑管理）
- 时光轴管理：支持拖拽排序、新增/编辑/删除事件
- 照片管理：批量上传、自定义描述、手动排序
- 音乐管理：支持多种音乐绑定方式

### 3. 核心功能
- 乐谱渲染：实时生成五线谱画册页面
- 3D翻页：流畅跟手的翻页动效
- 分享导出：生成高清海报、PDF文件

## 技术栈

- **前端框架**：微信小程序原生框架
- **开发语言**：TypeScript
- **云服务**：微信云开发
- **构建工具**：微信开发者工具

## 环境要求

- 微信开发者工具
- Node.js（用于TypeScript编译）
- 微信小程序开发账号

## 快速开始

1. 使用微信开发者工具打开项目
2. 配置小程序AppID（project.config.json中已配置）
3. 开启云开发环境
4. 编译运行小程序

## 设计规范

- **主色调**：深墨蓝背景、暖金色（#C9A96E）为主强调色
- **辅助色**：奶白色、玫瑰金、珊瑚粉、薰衣草紫
- **设计风格**：古典浪漫 + 数字精致
- **交互体验**：3D翻页动效、轻量化Toast提示

## 性能要求

- 首屏加载时长 ≤ 2s
- 单页乐谱渲染耗时 ≤ 500ms
- 低端机型自动降级为2D翻页

## 安全要求

- 上传文件限制：图片≤10MB、音频≤20MB
- 本地数据加密存储
- 调用微信官方授权接口

## 开发说明

- 使用TypeScript进行开发
- 遵循微信小程序开发规范
- 组件化开发，提高代码复用性
- 注重性能优化和用户体验
